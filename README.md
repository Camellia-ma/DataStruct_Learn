##### 一、项目介绍

该项目主要用于学习经典数据结构算法，例如线性表、树、图、堆等，项目目录结构为：
```
DataStruct
│   .gitignore             // 配置git仓库忽略文件
│   CMakeLists.txt         // 项目配置文件
├───.idea                  // IDE配置文件
├───cmake-build-debug      // Debug配置文件
├───runtime                // 存放编译后生成的可执行文件
└───src                    // 存放源代码
```
主要使用的编译器为: gcc version 8.1.0 (x86_64-posix-seh-rev0, Built by MinGW-W64 project)


#### 二、注意事项

使用此CMakeList时，若要新建C/C++语言文件，请按照以下步骤：
1. 右键src目录(如果没有src目录，请先创建)——新建——C/C++源文件
2. 在弹出的对话框中，输入文件名(仅允许英文小写及下划线，不要出现空格)，后缀为 ".c(C语言文件)"或".cpp(C++文件)"，切记不要勾选“添加到目标”，点击确定。
3. 点击 左上角横线——文件——重新加载CMake项目 。

**[注]**：未重载前，进入文件可能会有“不属于任何项目目标”的警告，重载后即会消失。
若此警告未消失，请检查文件是否在src目录下，以及文件名是否符合规范。
